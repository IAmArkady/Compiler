<h2>Цели и задачи программного продукта</h2>
&nbsp;&nbsp;<b>1)</b> Разработать компилятор для трансляции подмножества языка <i>PascalABC</i> в язык <i>Си</i>, используя инструменты <i>Flex</i> и <i>Bison</i> для создания лексических и синтаксических анализаторов; <br>
&nbsp;&nbsp;<b>2)</b> Обеспечить корректное преобразование синтаксических конструкций и операций подмножества исходного языка в эквивалентный код на языке <i>Си</i>, сохраняя функциональную эквивалентность программы.<br>
<h2>Описание исходных файлов</h2>
В папке <b>"share"</b> находятся вспомогательные файлы, документация и шаблоны для генерации кода. <br>
В папке <b>"bin"</b> находятся исполняемые файлы для запуска инструментов. Основные файлы в данной папке: <br>
&nbsp;&nbsp;∘ <b>flex.exe</b> - исполняемый файл, который запускает программу для генерации лексических анализаторов на основе входного файла lexer.l; <br>
&nbsp;&nbsp;∘ <b>bison</b> - исполняемый файл, который запускает программу для генерации синтаксических анализаторов на основе входного файла parser.y; <br>
&nbsp;&nbsp;∘ <b>lexer.l</b> - файл, содержащий правила для лексического анализатора <i>flex</i>, который идентифицирует заданные токены в тексте; <br>
&nbsp;&nbsp;∘ <b>parser.y</b> - файл, содержащий правила для синтаксического анализатора <i>bison</i>, в котором определяются грамматика языка и действия, выполняемые при распознавании конструкций, используя найденные токены анализатором <i>flex</i>;<br> 
&nbsp;&nbsp;∘ <b>start.bat</b> - сценарий командной строки, который автоматизирует процесс компиляции, запуская <i>flex</i> для генерации лексического анализатора из файла <i>lexer.l</i>, затем <i>bison</i> для генерации синтаксического анализатора из файла <i>parser.y</i> и компилятор <i>gcc</i> для сборки всех сгенерированных и исходных файлов в исполняемую программу <i>program.exe</i><br><br>
