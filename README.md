<h2>Цели и задачи программного продукта</h2>
&nbsp;&nbsp;<b>1)</b> Разработать компилятор для трансляции подмножества языка <i>PascalABC</i> в язык <i>Си</i>, используя инструменты <i>Flex</i> и <i>Bison</i> для создания лексического и синтаксического анализаторов; <br>
&nbsp;&nbsp;<b>2)</b> Обеспечить корректное преобразование синтаксических конструкций и операций подмножества исходного языка в эквивалентный код на языке <i>Си</i>, сохраняя функциональную эквивалентность программы.<br>
<h2>Описание исходных файлов</h2>
В папке <b>"share"</b> размещены вспомогательные файлы, документация и шаблоны для генерации кода. <br>
В папке <b>"bin"</b> размещены исполняемые файлы для запуска инструментов. Основные файлы в данной папке: <br>
&nbsp;&nbsp;∘ <b>flex.exe</b> - исполняемый файл, для запуска программы генерации лексических анализаторов на основе входного файла lexer.l; <br>
&nbsp;&nbsp;∘ <b>bison</b> - исполняемый файл,для запуска программы генерации синтаксических анализаторов на основе входного файла parser.y; <br>
&nbsp;&nbsp;∘ <b>lexer.l</b> - файл, содержащий правила для лексического анализатора <i>flex</i>, который идентифицирует заданные токены в тексте; <br>
&nbsp;&nbsp;∘ <b>parser.y</b> - файл, содержащий правила для синтаксического анализатора <i>bison</i>, в котором определяются грамматика языка и действия, выполняемые при распознавании конструкций с использованием токенов, найденных анализатором <i>flex</i>;<br> 
&nbsp;&nbsp;∘ <b>start.bat</b> - сценарий командной строки, который автоматизирует процесс компиляции, запуская <i>flex</i> для генерации лексического анализатора из файла <i>lexer.l</i>, затем <i>bison</i> для генерации синтаксического анализатора из файла <i>parser.y</i> и компилятор <i>gcc</i> для сборки всех сгенерированных и исходных файлов в исполняемую программу <i>program.exe</i><br><br>

<h2>Компиляция и запуск программы</h2>
&nbsp;&nbsp;В файле <i>lexer.l</i> содержится код для лексического анализатора <b>flex</b>, где на языке Си описаны правила для поиска лексем с помощью регулярных выражений и действий с ними.
С помощью программы <b>flex.exe</b> происходит компиляция файла <i>lexer.l</i>, которая генерирует код на язык Си, содержащийся в файле <b>lex.yy.c</b>.<br><br>
&nbsp;&nbsp;В файле <i>parser.y</i> содержится код для синтаксического анализатора <b>bison</b>, где для описания правил языка использована укорачивающая контекстно свободная грамматика, работающая с цепочкой токенов, выданных лексическим анализатором <b>flex</b>. Для разбора данной грамматики используется метод рекурсивного спуска. С помощью программы <b>bison.exe</b> происходит компиляция файла <i>bison.y</i>, которая генерирует код на язык Си, содержащийся в файле <b>parser.tab.c</b>.<br><br>
&nbsp;&nbsp;Для запуска компиляции программы необходимо из папки <i>bin</i> запустить <i>start.bat</i> и дождаться самой компиляции. После успешной компиляции в папке <i>bin</i> появится исполняемый файл <i>program.exe</i>.
